# 📊 AI Project 1: Customer Churn Prediction Using Machine Learning

## 📌 Project Overview

Customer churn is one of the biggest challenges for businesses. This project uses Machine Learning to predict whether a customer is likely to leave a company based on customer demographics, account information, and service usage.

The project demonstrates a complete Machine Learning workflow including data preprocessing, exploratory data analysis, feature engineering, model training, evaluation, and visualization.

---

# 🎯 Objectives

- Predict customer churn accurately.
- Compare multiple Machine Learning algorithms.
- Identify important factors influencing customer churn.
- Build a reusable Machine Learning pipeline.

---
